# Online Judge - 项目概述

> 最后更新：2026-03-17

## 1. 项目简介

Online Judge 是一个现代化的在线判题系统，用户可以提交代码解决算法题目，系统在 Docker 沙箱中自动编译、运行并判定结果。

项目核心价值：
- **安全的代码执行**：Docker 容器隔离，无网络访问，资源受限
- **真实可用的题库**：内置 23 道经典算法题，支持批量导入

## 2. 已实现功能

### 2.1 用户系统
- 注册 / 登录 / JWT 认证（Bearer Token）
- 三级角色权限：`user` / `admin` / `super_admin`
- 个人中心页面

### 2.2 题目系统
- 题目 CRUD（含 Markdown 描述、KaTeX 数学公式渲染）
- 测试用例在线管理（增删改查、样例标记）
- 难度筛选（easy / medium / hard）、标签筛选、全文搜索、分页
- 管理员题目编辑页（实时 Markdown 预览）

### 2.3 判题系统
- Docker 容器沙箱（`network_mode=none`，CPU / 内存限制）
- Celery + Redis 异步任务队列
- 支持 5 种语言（全部验证通过）：Python 3.11 / C++ (GCC 13) / Java 17 / Go 1.21 / JavaScript (Node 20)
- 判题结果：AC / WA / TLE / MLE / RE / CE

### 2.4 提交系统
- 代码提交后前端实时轮询（1.5s 间隔，最多 30 次，终态自动停止）
- 提交记录列表（分页、按题目 / 语言 / 状态筛选）
- 提交详情页（代码高亮、每个测试用例通过情况、错误信息）

### 2.5 竞赛系统
- 竞赛 CRUD（管理员），支持 ACM / OI 计分规则
- 公开 / 私有 / 密码赛三种访问模式
- 竞赛报名、题目管理、实时排行榜（分页）

### 2.6 题库导入
- `POST /api/v1/problems/import` 批量导入 API（仅管理员）
- 命令行脚本 `src/backend/scripts/import_problems.py`（支持 `--force` 覆盖）
- 内置 `classic_problems.json`（23 道：简单 8 / 中等 14 / 困难 1）
- 前端拖拽上传 / 粘贴 JSON 导入界面

## 5. 快速启动

```bash
# 1. 启动 Redis
docker run -d -p 6379:6379 redis:7-alpine

# 2. 后端（新终端）
cd src/backend && source .venv/bin/activate
uvicorn main:app --port 8000 --reload

# 3. Celery Worker（新终端）
cd src/judger
celery -A celery_app worker -Q judge,celery -c 2 --loglevel=info

# 4. 前端（新终端）
cd src/frontend && bun run dev
```

详细说明见根目录 `README.md`。
