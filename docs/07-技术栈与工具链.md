> 最后更新：2026-03-17

## 1. 最终技术选型

| 层级 | 技术 | 选型理由 |
|------|------|---------|
| 后端框架 | **FastAPI** 0.109+ | 原生 async，自动 Swagger 文档，Pydantic 验证 |
| ORM | SQLAlchemy 2.0 (async) + Alembic | 异步支持完善，类型友好 |

| 数据库（生产规划） | PostgreSQL 15+ | 完整事务支持，可扩展 |
| 消息队列 | Redis 7 + Celery 5 | 判题任务异步化，水平扩展方便 |
| 判题沙箱 | Docker | 容器隔离，多语言支持，macOS 兼容 OrbStack |
| 前端框架 | React 18 + TypeScript + Vite 5 | 生态丰富，极速 HMR |
| UI 组件库 | Ant Design 5 + TailwindCSS 3 | 企业级组件 + utility-first CSS |
| 状态管理 | Zustand 4 | 轻量，TypeScript 友好 |
| 代码编辑器 | Monaco Editor 4 | VS Code 同款内核 |
| Python 包管理 | **uv** | pip 的 10–100× 速度，内置 venv 管理 |
| 前端包管理 | **bun** | npm 的 20–30× 速度，内置运行时 |

## 2. Python 工具链：uv

### 2.1 安装

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### 2.2 常用命令

```bash
cd src/backend

uv venv .venv              # 创建虚拟环境
source .venv/bin/activate  # 激活环境
uv sync                    # 安装所有依赖（from pyproject.toml）
uv add fastapi             # 添加运行时依赖
uv add --dev ruff pytest   # 添加开发依赖
uv run python main.py      # 无需激活直接运行
```

### 2.3 项目配置（`pyproject.toml` 关键片段）

```toml
[project]
name = "online-judge-backend"
requires-python = ">=3.11"
dependencies = [
    "fastapi>=0.109",
    "uvicorn[standard]",
    "sqlalchemy[asyncio]>=2.0",
    "alembic",
    "aiosqlite",
    "celery[redis]",
    "docker",
    "python-jose[cryptography]",
    "passlib[bcrypt]",
    "pydantic-settings",
    "python-multipart",
]

[tool.ruff]
line-length = 100
select = ["E", "F", "I"]
```

## 3. 前端工具链：bun

### 3.1 安装

```bash
curl -fsSL https://bun.sh/install | bash
```

### 3.2 常用命令

```bash
cd src/frontend

bun install       # 安装依赖（比 npm install 快 20-30x）
bun run dev       # 启动开发服务器（:5173）
bun run build     # 构建生产包（输出 dist/）
bun run lint      # ESLint 检查
bun run preview   # 预览生产构建
```

## 5. 数据库迁移：Alembic

```bash
cd src/backend
source .venv/bin/activate

alembic revision --autogenerate -m "add_xxx_table"  # 自动生成迁移
alembic upgrade head                                 # 执行迁移
alembic downgrade -1                                 # 回退一步
alembic current                                      # 查看当前版本
alembic history                                      # 查看历史
```

迁移文件位于 `src/backend/alembic/versions/`。
