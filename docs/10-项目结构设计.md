## 6. 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| Python 文件 | snake_case | `docker_judge_service.py` |
| Python 类 | PascalCase | `DockerJudgeService` |
| Python 函数 | snake_case | `submit_judge_task` |
| TypeScript 文件 | PascalCase（组件）/ camelCase（其他） | `ProblemDetail/index.tsx` |
| TypeScript 接口 | PascalCase | `ContestListItem` |
| API 路由 | kebab-case | `/test-cases/{id}` |
| 数据库表名 | snake_case（复数） | `contest_participants` |
