# Online Judge 系统 - 前端设计

## 1. 技术栈

### 1.1 核心框架
- **React 18+**: 前端框架
- **TypeScript**: 类型安全
- **Vite**: 构建工具

### 1.2 UI 框架
- **Ant Design** 或 **Material-UI**: 组件库
- **TailwindCSS**: 样式工具
- **Lucide React**: 图标库

### 1.3 状态管理
- **Zustand** 或 **Redux Toolkit**: 全局状态管理
- **React Query**: 服务端状态管理

### 1.4 路由
- **React Router v6**: 客户端路由

### 1.5 代码编辑器
- **Monaco Editor**: VS Code 同款编辑器
- 支持语法高亮、自动补全、主题切换

### 1.6 其他工具
- **Axios**: HTTP 客户端
- **Day.js**: 时间处理
- **Recharts**: 数据可视化
- **React Markdown**: Markdown 渲染

## 2. 页面结构

### 2.1 整体布局

```
┌─────────────────────────────────────────────────────────┐
│                      顶部导航栏                          │
│  Logo | 题库 | 竞赛 | 讨论 | 排名 | [搜索] | 登录/用户   │
└─────────────────────────────────────────────────────────┘
│                                                          │
│                      主内容区                            │

│                      底部信息                            │
│              © 2024 OJ | 关于 | 帮助                     │
└─────────────────────────────────────────────────────────┘
```

### 2.2 页面列表

#### 公共页面
- **首页** (`/`): 系统介绍、统计数据、最新动态
- **题库** (`/problems`): 题目列表
- **题目详情** (`/problems/:id`): 题目描述、提交代码
- **提交记录** (`/submissions`): 所有提交记录
- **提交详情** (`/submissions/:id`): 单个提交详情
- **竞赛列表** (`/contests`): 所有竞赛
- **竞赛详情** (`/contests/:id`): 竞赛题目、排行榜
- **讨论区** (`/discussions`): 讨论列表
- **讨论详情** (`/discussions/:id`): 讨论内容和评论
- **排行榜** (`/ranklist`): 用户排名

#### 用户页面
- **登录** (`/login`): 用户登录
- **注册** (`/register`): 用户注册
- **个人中心** (`/user/:id`): 用户信息、统计、提交历史
- **设置** (`/settings`): 个人设置

#### 管理员页面
- **管理后台** (`/admin`): 仪表盘
- **题目管理** (`/admin/problems`): 题目 CRUD
- **用户管理** (`/admin/users`): 用户管理
- **竞赛管理** (`/admin/contests`): 竞赛管理
- **系统设置** (`/admin/settings`): 系统配置

### 5.1 全局状态

```tsx
interface GlobalState {
  user: User | null;
  token: string | null;
  theme: 'light' | 'dark';
  language: string;
}
```

### 5.2 用户认证

```tsx
const useAuthStore = create((set) => ({
  user: null,
  token: localStorage.getItem('token'),
  login: (token, user) => {
    localStorage.setItem('token', token);
    set({ token, user });
  },
  logout: () => {
    localStorage.removeItem('token');
    set({ token: null, user: null });
  },
}));
```
