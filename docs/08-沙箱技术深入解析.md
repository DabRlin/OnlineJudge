# Online Judge 系统 - 沙箱技术深入解析

## 1. 沙箱技术概述

### 1.1 什么是沙箱？

沙箱（Sandbox）是一种安全机制，用于在受限的环境中执行不受信任的代码，防止恶意代码对系统造成破坏。

### 1.2 为什么 OJ 需要沙箱？

在线判题系统需要执行用户提交的任意代码，存在以下安全风险：

**安全威胁：**
- 🔴 **恶意代码**：删除文件、修改系统配置
- 🔴 **资源耗尽**：无限循环、内存炸弹、fork 炸弹
- 🔴 **信息泄露**：读取测试数据、其他用户代码
- 🔴 **网络攻击**：发起 DDoS、扫描内网
- 🔴 **权限提升**：利用漏洞获取 root 权限

**沙箱的作用：**
- ✅ 进程隔离
- ✅ 资源限制（CPU、内存、磁盘、网络）
- ✅ 系统调用过滤
- ✅ 文件系统隔离
- ✅ 网络隔离

## 2. 沙箱技术方案对比

### 2.1 主流沙箱技术

| 技术 | 隔离级别 | 性能开销 | 复杂度 | 适用场景 |
|------|---------|---------|--------|---------|
| **Docker** | 容器级 | 低 | 中 | ✅ **推荐** |
| **LXC** | 容器级 | 低 | 高 | 大规模部署 |
| **虚拟机** | 系统级 | 高 | 中 | 高安全要求 |
| **ptrace** | 进程级 | 中 | 高 | 精细控制 |
| **seccomp** | 系统调用级 | 极低 | 高 | 配合其他技术 |
| **cgroup** | 资源级 | 极低 | 低 | 资源限制 |
| **namespace** | 进程级 | 极低 | 中 | 进程隔离 |

### 2.2 为什么选择 Docker？

**优势：**
- ✅ 完善的隔离性（namespace + cgroup + seccomp）
- ✅ 易于使用和维护
- ✅ 丰富的生态系统
- ✅ 性能开销低（相比虚拟机）
- ✅ 支持多种语言环境
- ✅ 可移植性强

**劣势：**
- ⚠️ 需要 root 权限（可用 rootless 模式）
- ⚠️ 共享内核（理论上存在逃逸风险）

## 3. Docker 沙箱深入解析

### 3.1 Docker 的核心技术

Docker 沙箱基于 Linux 内核的三大技术：

#### 3.1.1 Namespace（命名空间）

提供进程级别的隔离：

```
┌─────────────────────────────────────────────────────────┐
│ Namespace 类型                                           │
├─────────────────────────────────────────────────────────┤
│ PID    - 进程 ID 隔离（容器内 PID 1 独立）               │
│ NET    - 网络隔离（独立的网络栈）                        │
│ MNT    - 文件系统挂载点隔离                              │
│ UTS    - 主机名和域名隔离                                │
│ IPC    - 进程间通信隔离                                  │
│ USER   - 用户和用户组隔离                                │
│ CGROUP - cgroup 根目录隔离                               │
└─────────────────────────────────────────────────────────┘
```

**示例：PID Namespace**


#### 3.1.2 Cgroup（控制组）

提供资源限制和监控：

```
┌─────────────────────────────────────────────────────────┐
│ Cgroup 子系统                                            │
├─────────────────────────────────────────────────────────┤
│ cpu        - CPU 使用限制                                │
│ memory     - 内存使用限制                                │
│ blkio      - 块设备 I/O 限制                             │
│ cpuset     - CPU 核心绑定                                │
│ devices    - 设备访问控制                                │
│ freezer    - 暂停/恢复进程                               │
│ net_cls    - 网络流量分类                                │
│ pids       - 进程数量限制                                │
└─────────────────────────────────────────────────────────┘
```

**Cgroup 配置示例：**

```bash
# 查看容器的 cgroup 配置

# 手动配置 cgroup（底层实现）

# /sys/fs/cgroup/cpu/docker/<container_id>/cpu.cfs_quota_us
```

#### 3.2.2 资源限制配置

```python
def get_resource_limits(language: str, problem_config: dict):
    """根据语言和题目配置生成资源限制"""
    
    # 基础时间限制（毫秒）
    base_time_limit = problem_config.get('time_limit', 1000)
    
    # 语言系数
    language_multiplier = {
        'c': 1.0,
        'cpp': 1.0,
        'rust': 1.0,
        'go': 1.5,
        'java': 2.0,
        'python': 3.0,
        'javascript': 2.0,
        'ruby': 3.0,
    }
    
    multiplier = language_multiplier.get(language, 2.0)
    time_limit_ms = int(base_time_limit * multiplier)
    
    # 内存限制（字节）
    base_memory = problem_config.get('memory_limit', 256)  # MB
    memory_limit = base_memory * 1024 * 1024
    
    # Java 需要更多内存
    if language == 'java':
        memory_limit = max(memory_limit, 512 * 1024 * 1024)
    
    return {
        # CPU 限制
        'cpu_period': 100000,  # 100ms
        'cpu_quota': time_limit_ms * 1000,  # 转换为微秒
        
        # 内存限制
        'mem_limit': memory_limit,
        'memswap_limit': memory_limit,  # 禁用 swap
        
        # 磁盘 I/O 限制
        'blkio_weight': 500,  # 默认权重
        
        # 时间限制（用于 timeout）
        'timeout': time_limit_ms / 1000 + 5,  # 额外 5 秒缓冲
    }
```

#### 3.2.3 完整的判题容器配置

```python
import docker
from typing import Dict, Any

class JudgeContainer:
    def __init__(self, language: str, problem_id: int):
        self.client = docker.from_env()
        self.language = language
        self.problem_id = problem_id
        
    def create_container(
        self,
        code: str,
        test_input: str,
        time_limit: int,
        memory_limit: int
    ) -> Dict[str, Any]:
        """创建判题容器"""
        
        # 选择镜像
        image = f"oj-judge/{self.language}:latest"
        
        # 资源限制
        resources = get_resource_limits(self.language, {
            'time_limit': time_limit,
            'memory_limit': memory_limit
        })
        
        # 创建容器
        container = self.client.containers.create(
            image=image,
            
            # 命令
            command=self._get_run_command(),
            
            # 基础配置
            **DOCKER_BASE_CONFIG,
            
            # 资源限制
            **resources,
            
            # 环境变量
            environment={
                'LANG': 'C.UTF-8',
                'LC_ALL': 'C.UTF-8',
                'TIME_LIMIT': str(time_limit),
                'MEMORY_LIMIT': str(memory_limit),
            },
            
            # 挂载代码和测试数据
            volumes={
                f'/tmp/judge/{self.problem_id}/code': {
                    'bind': '/workspace/code',
                    'mode': 'ro'
                },
                f'/tmp/judge/{self.problem_id}/input': {
                    'bind': '/workspace/input',
                    'mode': 'ro'
                },
                f'/tmp/judge/{self.problem_id}/output': {
                    'bind': '/workspace/output',
                    'mode': 'rw'
                },
            },
            
            # 工作目录
            working_dir='/workspace',
            
            # 自动删除
            auto_remove=False,  # 需要获取日志后再删除
            
            # 标签（用于管理）
            labels={
                'app': 'oj-judge',
                'language': self.language,
                'problem_id': str(self.problem_id),
            },
        )
        
        return container
    
    def _get_run_command(self) -> list:
        """获取运行命令"""
        commands = {
            'cpp': ['/bin/sh', '-c', './main < /workspace/input > /workspace/output'],
            'python': ['/bin/sh', '-c', 'python3 main.py < /workspace/input > /workspace/output'],
            'java': ['/bin/sh', '-c', 'java Main < /workspace/input > /workspace/output'],
            'go': ['/bin/sh', '-c', './main < /workspace/input > /workspace/output'],
        }
        return commands.get(self.language, ['/bin/sh'])
```

## 4. 实战：完整的判题流程

### 4.1 判题流程实现

```python
# judge_worker/judge.py

import docker
import time
import os
import tempfile
from pathlib import Path
from typing import Dict, Tuple

class Judge:
    def __init__(self):
        self.client = docker.from_env()
        
    def judge_submission(
        self,
        submission_id: int,
        problem_id: int,
        language: str,
        code: str,
        test_cases: list
    ) -> Dict:
        """完整的判题流程"""
        
        # 1. 创建工作目录
        work_dir = self._create_work_directory(submission_id)
        
        try:
            # 2. 编译代码（如需要）
            if language in ['c', 'cpp', 'java', 'go']:
                compile_result = self._compile_code(
                    work_dir, language, code
                )
                if not compile_result['success']:
                    return {
                        'status': 'CE',
                        'error': compile_result['error']
                    }
            else:
                # 解释型语言，直接写入代码
                self._write_code(work_dir, language, code)
            
            # 3. 运行测试点
            results = []
            for i, test_case in enumerate(test_cases):
                result = self._run_test_case(
                    work_dir,
                    language,
                    test_case,
                    i
                )
                results.append(result)
                
                # 如果不是 AC，可以选择提前终止
                if result['status'] != 'AC':
                    break
            
            # 4. 汇总结果
            final_result = self._aggregate_results(results)
            
            return final_result
            
        finally:
            # 5. 清理工作目录
            self._cleanup(work_dir)
    
    def _create_work_directory(self, submission_id: int) -> Path:
        """创建工作目录"""
        work_dir = Path(f"/tmp/judge/{submission_id}")
        work_dir.mkdir(parents=True, exist_ok=True)
        
        # 创建子目录
        (work_dir / "code").mkdir(exist_ok=True)
        (work_dir / "input").mkdir(exist_ok=True)
        (work_dir / "output").mkdir(exist_ok=True)
        
        return work_dir
    
    def _compile_code(
        self,
        work_dir: Path,
        language: str,
        code: str
    ) -> Dict:
        """编译代码"""
        
        # 写入源代码
        src_file = self._write_code(work_dir, language, code)
        
        # 编译配置
        compile_commands = {
            'c': f'gcc {src_file} -o {work_dir}/code/main -O2 -std=c11 -lm -Wall',
            'cpp': f'g++ {src_file} -o {work_dir}/code/main -O2 -std=c++17 -lm -Wall',
            'java': f'javac -encoding UTF-8 {src_file}',
            'go': f'go build -o {work_dir}/code/main {src_file}',
        }
        
        compile_cmd = compile_commands.get(language)
        if not compile_cmd:
            return {'success': False, 'error': 'Unsupported language'}
        
        # 创建编译容器
        try:
            container = self.client.containers.run(
                image=f'oj-judge/{language}:latest',
                command=f'/bin/sh -c "{compile_cmd}"',
                volumes={
                    str(work_dir): {'bind': '/workspace', 'mode': 'rw'}
                },
                working_dir='/workspace',
                network_disabled=True,
                mem_limit='512m',
                cpu_quota=100000,
                remove=True,
                stdout=True,
                stderr=True,
            )
            
            return {'success': True}
            
        except docker.errors.ContainerError as e:
            return {
                'success': False,
                'error': e.stderr.decode('utf-8')[:1000]  # 限制错误信息长度
            }
    
    def _write_code(self, work_dir: Path, language: str, code: str) -> Path:
        """写入代码文件"""
        filenames = {
            'c': 'main.c',
            'cpp': 'main.cpp',
            'python': 'main.py',
            'java': 'Main.java',
            'go': 'main.go',
            'javascript': 'main.js',
        }
        
        filename = filenames.get(language, 'main.txt')
        src_file = work_dir / "code" / filename
        
        with open(src_file, 'w', encoding='utf-8') as f:
            f.write(code)
        
        return src_file
    
    def _run_test_case(
        self,
        work_dir: Path,
        language: str,
        test_case: Dict,
        test_index: int
    ) -> Dict:
        """运行单个测试点"""
        
        # 写入测试输入
        input_file = work_dir / "input" / f"test_{test_index}.in"
        with open(input_file, 'w') as f:
            f.write(test_case['input'])
        
        # 输出文件
        output_file = work_dir / "output" / f"test_{test_index}.out"
        
        # 运行命令
        run_commands = {
            'c': './code/main',
            'cpp': './code/main',
            'python': 'python3 code/main.py',
            'java': 'java -cp code Main',
            'go': './code/main',
            'javascript': 'node code/main.js',
        }
        
        run_cmd = run_commands.get(language)
        
        # 创建运行容器
        try:
            # 获取资源限制
            limits = get_resource_limits(language, test_case)
            
            # 运行容器
            start_time = time.time()
            
            container = self.client.containers.run(
                image=f'oj-judge/{language}:latest',
                command=f'/bin/sh -c "{run_cmd} < input/test_{test_index}.in > output/test_{test_index}.out"',
                volumes={
                    str(work_dir): {'bind': '/workspace', 'mode': 'rw'}
                },
                working_dir='/workspace',
                **DOCKER_BASE_CONFIG,
                **limits,
                remove=True,
                stdout=True,
                stderr=True,
            )
            
            end_time = time.time()
            time_used = int((end_time - start_time) * 1000)  # 毫秒
            
            # 读取输出
            with open(output_file, 'r') as f:
                user_output = f.read()
            
            # 对比输出
            is_correct = self._compare_output(
                user_output,
                test_case['expected_output']
            )
            
            # 获取资源使用情况
            # 注意：这里简化了，实际需要从 cgroup 读取
            memory_used = 0  # KB
            
            return {
                'status': 'AC' if is_correct else 'WA',
                'time_used': time_used,
                'memory_used': memory_used,
                'test_index': test_index,
            }
            
        except docker.errors.ContainerError as e:
            # 运行时错误
            return {
                'status': 'RE',
                'error': e.stderr.decode('utf-8')[:500],
                'test_index': test_index,
            }
        except Exception as e:
            # 其他错误（可能是 TLE、MLE）
            error_msg = str(e)
            if 'timeout' in error_msg.lower():
                status = 'TLE'
            elif 'memory' in error_msg.lower():
                status = 'MLE'
            else:
                status = 'SE'
            
            return {
                'status': status,
                'error': error_msg[:500],
                'test_index': test_index,
            }
    
    def _compare_output(self, user_output: str, expected_output: str) -> bool:
        """对比输出"""
        # 标准对比：忽略行尾空格和文件尾空行
        user_lines = [line.rstrip() for line in user_output.strip().split('\n')]
        expected_lines = [line.rstrip() for line in expected_output.strip().split('\n')]
        
        return user_lines == expected_lines
    
    def _aggregate_results(self, results: list) -> Dict:
        """汇总结果"""
        if not results:
            return {'status': 'SE', 'error': 'No test results'}
        
        # 检查是否全部 AC
        all_ac = all(r['status'] == 'AC' for r in results)
        
        if all_ac:
            status = 'AC'
        else:
            # 找到第一个非 AC 的状态
            status = next(r['status'] for r in results if r['status'] != 'AC')
        
        # 计算最大时间和内存
        max_time = max((r.get('time_used', 0) for r in results), default=0)
        max_memory = max((r.get('memory_used', 0) for r in results), default=0)
        
        # 通过的测试点数
        passed = sum(1 for r in results if r['status'] == 'AC')
        
        return {
            'status': status,
            'time_used': max_time,
            'memory_used': max_memory,
            'test_cases_passed': passed,
            'test_cases_total': len(results),
            'details': results,
        }
    
    def _cleanup(self, work_dir: Path):
        """清理工作目录"""
        import shutil
        try:
            shutil.rmtree(work_dir)
        except Exception as e:
            print(f"Failed to cleanup {work_dir}: {e}")
```

```python
def monitor_container(container_id: str):
    """监控容器资源使用"""
    client = docker.from_env()
    container = client.containers.get(container_id)
    
    # 获取统计信息
    stats = container.stats(stream=False)
    
    # CPU 使用率
    cpu_delta = stats['cpu_stats']['cpu_usage']['total_usage'] - \
                stats['precpu_stats']['cpu_usage']['total_usage']
    system_delta = stats['cpu_stats']['system_cpu_usage'] - \
                   stats['precpu_stats']['system_cpu_usage']
    cpu_percent = (cpu_delta / system_delta) * 100.0
    
    # 内存使用
    memory_usage = stats['memory_stats']['usage']
    memory_limit = stats['memory_stats']['limit']
    memory_percent = (memory_usage / memory_limit) * 100.0
    
    return {
        'cpu_percent': cpu_percent,
        'memory_usage': memory_usage,
        'memory_percent': memory_percent,
    }
```

## 7. 性能优化

### 7.1 容器复用

```python
class ContainerPool:
    """容器池，复用容器以提高性能"""
    
    def __init__(self, size=10):
        self.pool = []
        self.size = size
        self._initialize_pool()
    
    def _initialize_pool(self):
        """初始化容器池"""
        client = docker.from_env()
        for _ in range(self.size):
            container = client.containers.create(
                image='oj-judge/cpp:latest',
                **DOCKER_BASE_CONFIG
            )
            self.pool.append(container)
    
    def get_container(self):
        """获取容器"""
        if self.pool:
            return self.pool.pop()
        else:
            # 池空了，创建新容器
            client = docker.from_env()
            return client.containers.create(
                image='oj-judge/cpp:latest',
                **DOCKER_BASE_CONFIG
            )
    
    def return_container(self, container):
        """归还容器"""
        # 清理容器状态
        container.stop()
        container.remove()
        
        # 创建新容器放回池中
        client = docker.from_env()
        new_container = client.containers.create(
            image='oj-judge/cpp:latest',
            **DOCKER_BASE_CONFIG
        )
        self.pool.append(new_container)
```
